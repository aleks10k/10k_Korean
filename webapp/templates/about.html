About the application

This application can help you if you learn to read korean.

It would be better if you already have spent a week or two learning the basic phonetics.

There is a simple theory page where you can listen to how letters or syllables sound.

The app is going to be extended for training not only basic reading but grammar as well.

How it works

Click the settings button on the application page, check question content you want to see and click the apply button.

Chosen set of questions is repeated endlessly until you change settings.

Easy level contains all the letters of the alphabet.

Medium level contains simple syllables that are of consonant + vowel composition.

Hard level consists of all korean syllables that have consonant + vowel + consonant (patchim) composition.


The progress bar reflects a percent of correctly answered questions.

It isn't reset if you close a browser or refresh the page. It's reset if you click the reset button.
